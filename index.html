Diminishing returns & incrementality

Response curves expose where the next dollar still works — and where it’s wasted.

Marginal pipeline per next $1k invested

The incrementality question: not “what did this channel get credit for” but “what would one more dollar do.” Reallocate toward the left.

Reading the curves

Correlation ≠ incrementality. Attribution credit (sections 02–03) tells you which touches appear on winning journeys. It cannot tell you what would have happened anyway.

These response curves model diminishing returns: each channel’s pipeline rises with spend but flattens past its saturation point. The slope at the current operating point is the marginal return — the honest basis for the next budget decision.

A rigorous read requires geo holdouts, PSA / ghost-ad tests, or conversion-lift experiments to estimate true incremental lift. The curves below are a planning prior, not proof. (See methodology doc.)
